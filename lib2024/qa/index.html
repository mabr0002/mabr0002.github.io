  10. Question:What is the use of name and value attributes? Give two examples

    Answer:The use of name attribute is to specify the name of an input element, two examples of this are; <input type="checkbox" name="places" id="germany" value="germany">, and, <input type="password" name="pass" id="pass" required>. The use of a value attribute is to define the value for an imput field, two examples of this are; <input type="email" name="email" id="email" value="example@g.com" readonly>, and, <input type="submit" value="Submit">.

  12. Question:How would you link an external style sheet? Give example.

    Answer:You would link an external style sheet by starting with a link tag, adding the relationship between the two documents, the type of document being linked (in this case a style sheet), and then how the index page can access the style sheet with href. An example of this is; <link rel="stylesheet" type="text/css" href="css/style.css">
