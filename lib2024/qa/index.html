QA for LIB2024

  1. Question:What are three components of web?

    Answer:Three components of web are HTML, CSS, and JavaScript, three coding languages.

  2. Question:Define version control and give one example

    Answer:Version control is a system that tracks changes that you make to a file/document, and allows you to view previous changes made.
    Example:An example of this is the undo, redo buttons on word. This allows you to undo and redo changes made to a document.

  3. Question:What tag is used to create the title of a web page? Give an example

    Answer:The tag used to create the title of a web page is the <title>. tag
    Example:<title> Title Of Web Page </title>

  4. Question:What tag is used to create a paragraph in html? Give an example.

    Answer:The tag used to create a paragraph in HTML is <p>.
    Example:<p> This is a paragraph </p>

  5. Question:Identify the odd one from these tags: <span>,<p>,<a>,<i> and state the reason.

    Answer:<i>, this is a tag for italics, which is used to create a style in the webpage. While <span>,<p>,<a>, are used to create the skeleton of the page.

  6. Question:What are the tags required to create a navigation list in HTML? Give one example with two navigation links (https://algonquincollege.com, https://google.com).

    Answer: The tags required to create a navigation list in HTML are <ul>, <li>, and, <a>.

  7. Question:What element is used to create images? Name its attributes.

    Answer: The element used to create an image is <img>. Its attributes include src, alt, height, and, width.

  8. Question:Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason

    Answer:<table>, <tr>, and <td> are all used in creating tables while <b> is used to make text bold.

  9. Question:Give an example of color picker input.

    Answer:An example of a colour picker imput is <option value="purple">Purple</option>, another example of using a colour picker imput would be <input type="color" value="#ff0000">'

  10. Question:What is the use of name and value attributes? Give two examples

    Answer:The use of name attribute is to specify the name of an input element, two examples of this are; <input type="checkbox" name="places" id="germany" value="germany">, and, <input type="password" name="pass" id="pass" required>. The use of a value attribute is to define the value for an imput field, two examples of this are; <input type="email" name="email" id="email" value="example@g.com" readonly>, and, <input type="submit" value="Submit">.

  11. Question:Give an example of inline style.

    Answer:An example of inline style is;

    <style type="text/css">

      h1,h2 {

      color: purple;

      text-align: center

    </style>

  12. Question:How would you link an external style sheet? Give example.

    Answer:You would link an external style sheet by starting with a link tag, adding the relationship between the two documents, the type of document being linked (in this case a style sheet), and then how the index page can access the style sheet with href. An example of this is; <link rel="stylesheet" type="text/css" href="css/style.css">
