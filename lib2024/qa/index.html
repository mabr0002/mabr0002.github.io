  1. Question:What are three components of web?

    Answer:Three components of web are HTML, CSS, and JavaScript, three coding languages.

  2. Question:Define version control and give one example

    Answer:Version control is a system that tracks changes that you make to a file/document, and allows you to view previous changes made.
    Example:
    An example of this is the undo, redo buttons on word. This allows you to undo and redo changes made to a document.

  3. Question:What tag is used to create the title of a web page? Give an example

    Answer:

    The tag used to create the title of a web page is the <title> tag
    Example:
    <title> Title Of Web Page </title>

  4. Question:What tag is used to create a paragraph in html? Give an example.

    Answer:

    The tag used to create a paragraph in HTML is <p>.
    Example:
    <p> This is a paragraph </p>
